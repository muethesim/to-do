# ToDo APP

A Simple flutter project to understand Hive.

## More

- Simple app to add, check, and delete daily to-do activities.
- Used Hive Database to manage data.

## Dependencies Used

- flutter_slidable: ^3.0.0
- hive: ^2.2.3
- hive_flutter: ^1.1.0

## Dev Dependencies Used

- hive_generator: ^2.0.1
- build_runner: ^2.4.6
